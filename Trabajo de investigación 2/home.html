¿Qué es Bootstrap?

Es un framework front-end utilizado para desarrollar aplicaciones web y sitios mobile first, o sea, con un layout que se adapta a la pantalla del dispositivo utilizado por el usuario.

El framework combina CSS y JavaScript para estilizar los elementos de una página HTML. Permite mucho más que, simplemente, cambiar el color de los botones y los enlaces.

Esta es una herramienta que proporciona interactividad en la página, por lo que ofrece una serie de componentes que facilitan la comunicación con el usuario, como menús de navegación, controles de página, barras de progreso y más.

¿Cuál es la funcionalidad de Bootstrap?

En pocas palabras bootstrap es una herramienta que facilita el diseño y el maquetado de una pagína web brindando bibliotecas de estructuras ya ordenadas con css y js listas para usar.

A demás de una gran cantidad de características como caruseles de fotos, menús, barras de navegacion, alertas, etc.

¿Cómo aplico Bootstrap a mi página?

Para poder aplicar Bootstrap, solo debes agregar los enlaces que puedes encontrar en la página oficial y utilizar el código que está organizado por componentes y clases predefinidas para facilitar el desarrollo web.
